# 图片优先分享海报设计

## 目标

分享海报不再用纵向票据结构压缩录音首帧。完整现场画面必须成为海报主体；Scene、Synth、Rhythm、BPM、Root、FX 参数全部删除。

## 版式

- 海报改为接近方形的 `1200×1080`，降低纵向冗余。
- 现场画面使用 `1120×700` 最大区域，完整等比放入，不裁切、不拉伸，并在区域内水平与垂直居中。
- 顶部只保留品牌、Player、Take 和时长。
- 底部只保留玩家身份、扫码提示和二维码。
- 分享弹窗恢复宽松布局，桌面端海报预览从 `280px` 放大到 `420px`；小屏继续单列自适应。

## 验收

- 16:10 和 16:9 首帧均完整显示。
- 海报中不存在 Scene、Synth、Rhythm、BPM、Root、FX 文本。
- 现场画面占海报宽度至少 90%，占海报有效视觉高度的主要部分。
- 桌面分享弹窗中的海报预览宽度为 420px，不再被固定在 280px。
