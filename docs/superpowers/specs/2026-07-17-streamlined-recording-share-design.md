# WAIC 双手乐队：录音、手势与分享流程收敛设计

日期：2026-07-17  
状态：已获用户确认

## 目标

把当前分散的“录音确认—拍照—照片确认—上传—分享”压缩为一次自然的演出闭环，同时修复确认页手势失效、画面变形/镜像、滤镜长期发闷、音色入口割裂和顶部状态栏遮挡。

## 已确认流程

主流程固定为：

`idle → 3/2/1 倒计时 → 自动截取录音首帧 → recording → stopping → review → uploading → shared`

- 录音开始后的首个可绘制帧自动成为海报画面，不再出现独立拍照倒计时、照片确认、重拍或“无照片继续”。
- 录音确认页保留试听、重录、下载、放弃；双手点赞直接等价于“确认并生成分享”，不再进入拍照页。
- 上传失败返回 review，保留录音与封面，允许再次提交。
- 重录会创建新的录制代次；旧的异步截图不得覆盖新录制的封面。
- 截图失败不阻断录音，海报使用抽象现场图形降级。

## 手势确认

根因不是识别模型失效，而是停止录音时的双手点赞消耗了共享 GestureLatch；进入 review 后锁仍未完成中立姿势重置，而且界面没有解释。

修复策略：

- 录音停止与 review 确认使用明确的阶段边界；进入 review 后启动新的确认门控。
- 要求约 500ms 中立姿势后重新举起双手点赞，防止同一次手势穿透两个阶段。
- review 显示“先放下双手 / 可以点赞确认 / 正在确认”的可见反馈与进度。
- 鼠标按钮始终可用，不让摄像头或手势状态成为阻塞条件。

## 录音首帧与画面方向

- 预热 html2canvas，录音真正开始后在首个 `requestAnimationFrame` 捕获页面合成画面。
- 海报使用页面实际宽高比裁切，不再先塞进固定 4:5 中间画布，从源头消除上下留黑和高度压扁。
- 摄像头图像按他人看到的正常方向输出；HUD 与网页文字保持正常可读。
- 合成时只取消视频层镜像，并按现有坐标系统单独镜像 WebGL 手势覆盖层，禁止翻转整张画布。

## 旋律音色

全局只保留一套旋律音色库，顺序为：DX7 E.PIANO、DX7 BRASS、DX7 MARIMBA、NEON PLUCK、NEON LEAD、ARCADE PULSE、ARCADE CRYSTAL、AFTERGLOW PAD、COASTAL PLUCK、BLUE HOUR KEYS、TAPE CHOIR。

- Scene 只控制序列、低音、调式、BPM 等编曲参数，不再隐式重置旋律音色。
- 握拳循环与高级控制面板下拉框调用同一个 `setTimbre` 入口。
- HUD 的 SYNTH、下拉框和实际合成器音色必须始终同步。
- 默认音色为 DX7 E.PIANO。

## 双手距离滤镜

距离采用“双掌中心距离 / 双手平均掌长”归一化，目标曲线：

- 距离不大于 1.25：LP 8%。
- 1.25 到 2.20：平滑升至 LP 82%。
- 2.20 到 2.75：平滑升至 LP 100%。
- 2.75 及以上：LP 100%。

使用 smoothstep 曲线、约 100ms 平滑和全开滞回：低于 2.55 才重新进入滤镜控制，拉远到 2.75 才完全打开。失去双手、手势被抑制或模式切换时恢复 100%。正常双手距离应稳定在 80–100%，只有刻意靠近才明显变闷。

## 顶部 HUD

- 顶部中心第一行仅放五个核心指标：Scene、Synth、Rhythm、BPM、Root。
- FX 独占下方一条紧凑效果带，展示 LP / DLY / GLT，不与“开始录音”动作区共用列宽。
- 录音、动作指引、显示模式和高级控制保持独立动作区；桌面与窄屏均不得互相遮挡。

## 海报与分享界面

海报采用“现场技术票据”视觉：横向现场窗口、Player/Take/时长、Scene/Synth/Rhythm/BPM/Root/FX 元数据、由本次录制标识和参数生成的独特轨迹、二维码及有效期。轨迹是本次演出指纹，不宣称为真实音频波形。

最终分享弹窗压缩到约 760–800px：左侧约 280px 海报，右侧为紧凑元数据、链接和复制/下载动作，消除大块空白与信息失衡。沿用深青、薄荷、青蓝和珊瑚橙的展览视觉语言。

## 验收标准

- 录音结束后没有独立拍照页面；确认后直接上传并进入分享。
- review 中重新做双手点赞可以可靠触发确认，并有明确重置提示。
- 海报首帧不压扁、不带上下黑带，人物方向正常，文字不反。
- Scene 切换不改变旋律音色；握拳、控制面板、HUD 三者同步。
- 正常双手距离 LP 为 80–100%，靠近逐渐变闷，拉远平滑回到 100%。
- 顶部 MIX FX 不再覆盖录音按钮。
- 分享弹窗紧凑，海报包含本次演出数据与唯一轨迹。
