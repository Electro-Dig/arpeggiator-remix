### 🔧 **技术实现细节**

#### **音阶映射逻辑**
```javascript
// 7音模式：大调音阶模式
const majorScalePattern = [0, 2, 4, 5, 7, 9, 11]; // C大调半音程间隔

// 度数到半音程转换
const octave = Math.floor(interval / 7);
const degree = ((interval % 7) + 7) % 7;
const semitone = octave * 12 + majorScalePattern[degree];
```
